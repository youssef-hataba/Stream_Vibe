# Stream-Vibe

Stream-Vibe is a movie review website built with React and Next.js. It allows users to explore and review movies, providing a platform for movie enthusiasts to share their opinions and discover new films.

## Features

- **User Reviews**: Users can submit reviews for their favorite movies.
- **Movie Listings**: Browse a list of movies with detailed information.
- **Responsive Design**: The website is designed to be mobile-friendly and accessible on all devices.
- **Server-Side Rendering**: Leveraging Next.js for optimized performance and SEO.

## Tech Stack

- **Frontend**: 
  - React
  - Next.js
  - Tailwind CSS

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) (v12 or later)
- [Git](https://git-scm.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:youssef-hataba/Stream-Vibe.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Stream-Vibe
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to see your app in action.

## Usage

- Visit the homepage to see a list of movies.
- Click on any movie to view details and leave a review.
- Explore user reviews and ratings to help decide what to watch next.
